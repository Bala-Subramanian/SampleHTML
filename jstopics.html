Arrow Function

Using Normal function

var getRegValue = function (p1){ return 10*p1; }
console.log(getRegValue(2));

Using Arrow function

const getArrowValue = (p1) => { return 10*p1; }
console.log(getArrowValue(3));

Filter Function

filter() does not execute the function for array elements without values

filter() does not change the original Array

Syntax: array.filter( function (fnValue*, index!,arr!), thisValue! )

var agesArray = [10,20,30,40];
const checkAdult = (age) => { return age>=18; }
const filterMethod = () => { return agesArray.filter(checkAdult); }
console.log(filterMethod());
